# 攝影機校正

## 攝影機校準教學

進行檢測之前，調整好攝影機高度、水平、距離(2M 以上)、角度

確保受測者，手舉高全身可以進去攝影捕捉畫面內，並保持人體大小適中且居於畫面中央且攝影機水平擺放

在設定開啟輔助線後
返回主畫面並進入檢測待攝影機載入後觀察畫面進行調整
